CPI Website Revamped

Client Project · WordPress Developer / UX Implementation

Project Background

This internal tool was developed in collaboration with the HR department to streamline recruitment. The project focused on building a custom Job Portal with essential features tailored for talent acquisition.

Problem Statement

The existing recruitment system was inefficient and not user-friendly for either HR personnel or applicants. Manual processing, unclear application forms, and lack of timely feedback slowed down hiring. The goal was to create a centralized, automated solution that simplified and clarified the process for both ends.

My Role

WordPress Developer – responsible for plugin development, custom application forms, email integration, and coordination with HR to implement necessary functionality.

Key Contributions

  • Developed custom WordPress plugins using PHP, HTML/CSS, and JavaScript.
  • Built application forms with complete logic, including email notifications to both candidates and HR.
  • Collaborated with the HR team to design features tailored to their recruitment needs.
  • Improved submission forms based on candidate feedback for better usability.
  • Provided ongoing maintenance and troubleshooting to ensure smooth daily operations.

Tools Used

WordPress, PHP, JavaScript, HTML/CSS, SMTP Email Integration

UX Considerations

Form fields were grouped logically and used placeholder hints and real-time validation to prevent input errors. The job portal navigation was designed to be clear, using plain language labels and feedback messages upon submission. I worked with HR to ensure that the interface was easy to understand—even for non-tech-savvy users.

Accessibility Considerations

Labels and form elements were semantically structured for screen reader compatibility. High-contrast text and button styling were used to improve readability. Required fields were clearly marked with asterisks and error highlights to aid user understanding.

Next Steps

Future improvements could include an applicant dashboard, file upload status indicators, and analytics for the HR team. A built-in scheduling tool for interviews and status-based email triggers would also enhance workflow automation.

Reflection

This project helped me strengthen my WordPress development skills, particularly in customizing functionality for specific team workflows. It also highlighted the importance of cross-functional collaboration and UX thinking in building tools that are not only functional, but also easy and pleasant to use.
