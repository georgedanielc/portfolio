WEBSITE REVAMPED

Client Project · WordPress Developer / UX Implementation

Project Background

Partnered with HR to create an internal recruitment portal, shifting from third-party job boards to a centralized, branded platform for better control and candidate experience.

Problem Statement

The old process was fragmented and manual, lacking structured forms, centralized data, and timely communication. The goal: build an in-house system that streamlined applications and automated essential steps.

My Role

WordPress Developer — built custom plugins, dynamic forms, and integrated email notifications while working closely with HR to meet their workflow needs.

Key Contributions

  • Developed custom plugins with PHP, JavaScript, and HTML/CSS.
  • Built application forms with validation and automated email alerts.
  • Implemented HR feedback for usability and streamlined backend tracking.
  • Provided post-launch refinements for improved performance.

Tools Used

WordPress, PHP, JavaScript, HTML/CSS, SMTP Email Integration

UX Considerations

Logical field grouping, real-time validation, and clear success/error states improved usability. Labels and navigation were designed for clarity with minimal training required.

Accessibility Considerations

Used semantic labels, tested color contrast, and combined text with color for error indicators to ensure screen reader compatibility and visual clarity.

Next Steps

Future improvements could include file upload tracking, HR analytics, and interview scheduling to enhance automation and reporting.

Reflection

This project strengthened my WordPress development skills and highlighted the impact of cross-functional collaboration in creating streamlined, user-friendly solutions.
