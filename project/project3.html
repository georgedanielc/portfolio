THIRD MUG COFFEE ORDERING UI

Concept Project · UI/UX Design

Project Background

A minimalistic mobile ordering experience for a fictional coffee brand, “Third Mug.” Focused on clean UI, simple interactions, and a smooth ordering flow.

Problem Statement

Most coffee apps are cluttered and slow for quick orders. This concept streamlines the process, removing distractions and guiding users through an efficient, focused flow.

My Role

UI Designer — handled wireframing, prototyping, and visual design of the ordering flow.

Tools Used

Figma

Prototype

Accessibility Considerations

High contrast colors, large readable fonts, and finger-friendly tap targets were prioritized. Visual clutter was reduced to improve focus and reduce cognitive load.

UX Considerations

The core focus was on reducing user effort in a mobile ordering flow. I prioritized large, tappable elements for easy thumb use, a linear ordering path to eliminate confusion, and clear confirmation states. Only the essential screens were included, removing clutter and letting users complete an order in just a few taps. Icons and labels were paired to support both recognition and clarity.

Next Steps

Designed a linear flow with large tappable elements for easy thumb use, minimal screens, and clear confirmation states. Paired icons with labels for faster recognition.

Reflection

Plan to conduct user testing with on-the-go customers to validate efficiency. Future features could include loyalty rewards and order history.
