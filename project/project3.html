Third Mug Coffee Ordering UI

Concept Project · UI Design

Project Background

A minimalistic mobile ordering experience designed for a fictional coffee brand, "Third Mug." The focus is on clean UI, intuitive interaction, and seamless user flow in a mobile context.

My Role

UI Designer — handled wireframing, prototyping, and visual design of the ordering flow.

Reflection

This concept challenged me to simplify decision-making in a mobile UI. I prioritized hierarchy, feedback, and touch targets for a smooth user journey from drink selection to checkout.
