LinkQ - Custom Link In Bio Page Generator

Live Project · Frontend Development & UI/UX Design

Project Background

Developed a responsive web application that lets users create a personalized “link in bio” page with customizable sections, colors, and live previews. Designed for creators, small businesses, and personal branding needs.

Problem Statement

Existing “link in bio” tools often limit customization or require subscriptions for premium features. I wanted to create a fast, free, and flexible alternative where users can fully control the design and content.

My Role

Full-Stack Designer & Developer — handled wireframing, UI design, responsive layout, live preview logic, Firebase integration, and deployment.

Tools Used

HTML, Tailwind CSS (CDN), Vanilla JavaScript, Firebase Firestore, Firebase Hosting, Lucide Icons

UX Considerations

Implemented real-time preview for instant feedback, structured sections for clear content organization, and ensured mobile-first design for better accessibility on small screens.

Accessibility Considerations

Used semantic HTML for structure, color pickers with sufficient contrast options, touch-friendly buttons, and keyboard-accessible navigation.

Reflection

This project strengthened my skills in dynamic UI creation, real-time DOM updates, and cloud database integration. It was an exciting challenge that blended UI design principles with practical frontend development.
