PERSONAL PORTFOLIO WEBSITE

Live Project · Frontend Development & UI Design

Project Background

Built a custom portfolio to showcase my design and frontend skills, reflecting a clean, minimalist style focused on clarity and interactivity.

Problem Statement

Template-based builders felt restrictive and lacked personality. I wanted a fast, lightweight, fully custom portfolio that represents me as both designer and developer.

My Role

Full-Stack Designer — handled layout, typography, responsive design, and JavaScript interactions from scratch.

Tools Used

HTML, CSS, JavaScript, Figma, Photoshop

UX Considerations

Structured content for easy scanning, optimized for mobile, and ensured color contrast met WCAG standards for readability.

Accessibility Considerations

Implemented semantic HTML, keyboard navigation, alt text for icons, and touch-friendly elements for inclusive usability.

Reflection

Building my portfolio reinforced best practices in performance, accessibility, and clean code. It was a rewarding challenge that showcases both my design sense and technical ability.
