PERSONAL PORTFOLIO WEBSITE

Live Project · Frontend Development & UI Design

Project Background

I built this portfolio to showcase my design and frontend development skills. It reflects my minimalist aesthetic and focus on clarity, interactivity, and structure.

Problem Statement

Portfolio builders felt restrictive, and template sites lacked personality. I wanted a portfolio that was lightweight, fast, fully custom, and represented me—both as a designer and a developer.

My Role

Full-Stack Designer — handled everything from layout, typography, color system, responsiveness, and vanilla JS interactions.

Tools Used

HTML, CSS, JavaScript, Figma, Photoshop

UX Considerations

I structured content to be scannable and prioritized accessibility. The layout adjusts smoothly on mobile, and the color system meets WCAG contrast requirements.

Accessibility Considerations

Semantic HTML, keyboard navigation, alt text for all icons, and responsive touch targets were all implemented to ensure usability for all users.

Next Steps

Reflection

Building my site helped reinforce best practices in accessibility, performance, and clean code. It was a rewarding challenge that reflects both my personal style and technical strengths.
