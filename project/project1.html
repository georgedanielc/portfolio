Design Rally 2025

Personal Project · UI/Visual Design

Project Background

This project explores a bold visual identity for a fictional design conference, “Design Rally 2025.” The goal was to experiment with energetic color palettes, dynamic type, and layout hierarchy.

My Role

Visual Designer, responsible for concept, layout, and presentation design.

Tools Used

Figma, Illustrator, Photoshop

Visuals

Reflection

Through this project, I explored how type and color can shape emotion and energy in an event setting. I learned to balance playfulness with readability, and I enjoyed pushing boundaries without real client constraints.
